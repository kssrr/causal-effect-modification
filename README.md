# causal-effect-modification

Replication data for my MSc thesis "Causal and Non-Causal Approaches to Treatment Effect Heterogeneity in Conjoint Experiments".

The code is in Quarto-Notebooks in the `R/`-folder. The rendered notebooks are in the `output/`-folder. If you decide to run the code yourself, be aware that it will try to import Tex-fonts you may not have installed on your system (you can simply skip that part in the beginning). Also be aware that results might differ slightly across runs, due to random processes (although I have set a seed wherever needed).

> **Abstract:** How can researchers identify causal treatment effect modification when the effect modifier is not randomized? Researchers often study heterogeneous treatment effects by comparing how the treatment operates across subgroups defined by pre-treatment characteristics. Because these characteristics are rarely randomized, observed differences in treatment effects may reflect covariate imbalance rather than genuine causal effect modification. This thesis addresses how to estimate causal treatment effect modification when the effect modifier of interest is observational. I propose using inverse probability weighting as a form of nonparametric preprocessing: a flexible model estimates propensity scores for the effect modifier, and a simple parametric model is then estimated on the weighted data. Weighting balances covariates across modifier groups, allowing differences in treatment effects to be interpreted as causal effect modification rather than only as observed heterogeneity. This approach combines flexible propensity score estimation with interpretable outcome modeling and reduces model dependence, and reliance on restrictive functional form assumptions. I conduct a simulation study to show that this approach is consistent, and illustrate its implementation using original survey experimental data on wartime victimization and intergroup trust collected in Bosnia and Herzegovina.
